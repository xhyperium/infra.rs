# yahoo — 测试策略

当前只执行本地 fixture 测试，不访问外部服务。

| 类别 | 覆盖内容 |
|---|---|
| 解析 | quote、chart、fx、search、未知字段和坏 UTF-8 |
| 领域映射 | 时间、单位、缺失值、重复身份和排序 |
| 安全 | secret、Cookie、完整 URL、原始响应不出现在错误或日志 |
| 合同失败 | 拒绝、挑战、配额和未知权限返回稳定错误 |

任何网络测试必须有书面授权、脱敏配置、独立人工批准和显式 CI 开关；不能通过“跳过”变成绿色证据。
