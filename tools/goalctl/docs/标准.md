# goalctl 标准文档

**包名**：`goalctl` v0.2.0 | **角色**：Goal 编译器 CLI | **层级**：Tool

---

## 1. 需求标准

### 目标

goalctl 提供最小化 Goal-to-Contract 编译能力，包含 doctor（诊断）、validate（验证）、compile（编译）三个子命令。

### 边界

- **包含**：doctor 诊断、validate 验证、compile 编译、fixtures + VERSION 单元测试
- **不包含**：完整权限平面、IDE 集成
- **依赖**：clap、serde、serde_json、serde_yaml、sha2、hex

---

## 2. 设计标准

### 架构

```text
goalctl
├── cmd/         — doctor/validate/compile 子命令
├── compiler/    — Goal → Contract 编译逻辑
├── validator/   — 验证逻辑
└── doctor/      — 诊断逻辑
```text

### CLI 结构

```text
goalctl doctor      — 检查环境/配置
goalctl validate    — 验证 Goal 文件
goalctl compile     — 编译 Goal → Contract
```text
