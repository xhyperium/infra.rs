# verifyctl 标准文档

**包名**：`verifyctl` v0.1.0 | **角色**：验证 CLI | **层级**：Tool

---

## 1. 需求标准

### 目标

verifyctl 提供最小化验证规划/执行/报告能力，包含 plan（规划）、execute（执行）、report（报告）三个子命令。

### 边界

- **包含**：plan（验证规划）、execute（执行验证）、report（生成报告）
- **不包含**：自动修复、CI/CD 紧密集成
- **依赖**：clap、serde、serde_json、sha2、evidence (optional, feature "with-evidence")

---

## 2. 设计标准

### 架构

```text
verifyctl
├── cmd/         — plan/execute/report 子命令
├── planner/     — 验证规划逻辑
├── executor/    — 验证执行逻辑
└── reporter/    — 报告生成逻辑
```text

### CLI 结构

```text
verifyctl plan      — 规划验证方案
verifyctl execute   — 执行验证
verifyctl report    — 生成验证报告
```text
