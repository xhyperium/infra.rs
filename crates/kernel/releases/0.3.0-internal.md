## 3. 消费者接入

```toml
# workspace 内（推荐）
kernel = { path = "crates/kernel", version = "0.3.0" }
```

```bash
cargo test -p kernel --all-targets
cargo run -p kernel --example basic
```

**支持矩阵**：Linux x86_64 · MSRV 1.85（见 `.agents/rules/support-matrix.md`）。

## 4. 明确不做

- `publish = true` / `cargo publish`
- 非 Linux 官方支持矩阵扩大
- archgate / monorepo 专属机控（**OOS**：本仓明确不移植，#164）
- 把 L1+L4 写成无条件「Production Ready」营销句
