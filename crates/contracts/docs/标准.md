# contracts 标准文档

**包名**：`contracts` v0.1.2 | **角色**：L3 合约层 trait 出口 | **层级**：L3 subset（非整体 Production Ready）

---

## 1. 需求标准

### 目标

contracts 定义跨层合约 trait（KeyValueStore、EventBus、MarketDataSource、ExecutionVenue 等），遵循 Additive Only 原则（R4）。

### 边界

- **包含**：合约 trait 定义、仅表达接线意图的 LiveContractProfile、形状校验 LiveHandles、准确命名的 kv/bus/tx helper
- **不包含**：具体实现、业务逻辑、传输层
- **依赖**：kernel、canonical、async-trait、bytes、futures-core

### 约束

- R4：Additive Only — trait 只能添加不能修改/删除方法
- L3 subset：先发 KV+Instr 子集
- trait 最小化设计

---

## 2. 设计标准

### 架构

```text
contracts
├── traits/      — KeyValueStore, EventBus, ExecutionVenue, MarketDataSource...
├── live/        — LiveContractProfile, LiveHandles
├── helpers/     — kv, bus, tx 辅助函数
└── docs/contracts/ — 各 trait 详细说明
```

### 核心 trait

| trait | 方法 | 说明 |
|-------|------|------|
| KeyValueStore | get/set/delete | KV 存储契约 |
| EventBus | publish/subscribe | 事件总线契约 |
| ExecutionVenue | place_order/cancel_order | 执行场所 |
| MarketDataSource | subscribe_quote/subscribe_trade | 市场数据 |
