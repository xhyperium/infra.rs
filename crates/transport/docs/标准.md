# transportx 标准文档

**包名**：`transportx` v0.1.2 | **角色**：L1 HTTP/WS 传输 | **层级**：L1 Internal Ready

---

## 1. 需求标准

### 目标

transportx 提供 HTTP/WS 传输能力：HttpDriver（HTTP 驱动）、WsConnector（WS 连接器）、TLS 配置、客户端池、代理配置。

### 边界

- **包含**：HttpDriver（基于 reqwest）、WsConnector（基于 tokio-tungstenite）、TlsConfig/TlsMode、HttpClientPool（工厂模式+错误回滚）、ProxyConfig
- **不包含**：完整生产 TLS/mTLS 产品、自定义协议
- **依赖**：reqwest、tokio-tungstenite、async-trait、bytes、kernel

---

## 2. 设计标准

### 架构

```text
transportx
├── http        — HttpDriver (reqwest 封装)
├── ws          — WsConnector (tokio-tungstenite)
├── tls         — TlsMode / TlsConfig
├── pool        — HttpClientPool（工厂 + 错误回滚）
└── proxy       — ProxyConfig
```
