## 变更摘要

- 新增 `ContractStoreSet`，固定注入正式 `KeyValueStore` / `EventBus` trait object。
- `Bootstrap`、`PlatformContext` 与 `AppContext` 增加 typed 注入/只读访问器。
- 固定摘要 Redis/NATS 组合实验从 `AppContext` 正式访问器完成 KV 与 pub/sub。

## 验证

```bash
cargo test -p bootstrap --all-targets
cargo clippy -p bootstrap --all-targets -- -D warnings
node scripts/storage-composition-conformance.mjs
```

## 未承诺

不提供动态 Service Locator、泛型 Repository 注册、跨资源事务或完整异步生命周期编排。
