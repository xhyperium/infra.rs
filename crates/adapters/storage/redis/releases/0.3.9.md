# redisx 0.3.9

## 范围

- `RedisMetricsSnapshot` + `RedisPool::metrics_snapshot`
- feature `pubsub`：`into_result_message_stream`（断线末尾一次 `Err`）

## 诚实边界

- 指标为进程内低基数累计，非 OTel/Prometheus exporter
- Pub/Sub 无可靠投递、无自动重连
- Cluster / Sentinel / TLS live 仍 OPEN
- **未**宣称 package stable

## 验证

```bash
cargo test -p redisx --lib
cargo test -p redisx --features pubsub --lib
cargo clippy -p redisx --all-targets --features pubsub -- -D warnings
```
