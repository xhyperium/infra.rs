# redisx 0.3.2

| 字段 | 值 |
|------|-----|
| 日期 | 2026-07-22 |
| 范围 | 内部 workspace（`publish = false`） |

## 变更摘要

- **生产入口接 resiliencx**：`RedisClient::with_retry_budget` 后，`get`/`set`/`delete`/`exists`/`expire`/`ttl`/`mget`/`mset` 经 `with_budget_async`。
- **显式 API**：`get_with_budget` / `set_with_budget` 始终走预算环。
- **async helpers**：`with_budget_async` / `with_budget_async_noop`（`resilience.rs`）。
- live I/O 仍仅 `tests/live_*`（`#[ignore]`）；默认 `--lib` 不碰外部 Redis。

## 验证

```bash
cargo test -p redisx --lib
cargo clippy -p redisx --all-targets -- -D warnings
```
