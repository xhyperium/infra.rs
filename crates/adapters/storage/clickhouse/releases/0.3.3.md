## 变更摘要（三轮加固：R1 负向验收 → R2 对抗回归 → R3 门禁收口）

- R1：`insert_json_each_row` / `insert_batch` 的表名与行结构校验路径补齐单测，并证明校验发生在任何网络请求**之前**；空 `rows` 短路成功路径补齐单测。
- R1：`query_rows` 的 TabSeparated 解析（跳过空行、按 tab 拆列）、`map_http_error` 的完整分支覆盖（404/57/81/5xx/403/未知 4xx）、`read_error_prefix` 的 4096 字节截断边界均补齐专项单测。
- R1：scaffold `ClickHouseAdapter` 的多事件累加、身份访问器与 `local()` 默认值补齐单测。
- R2：背压边界对抗测试——`max_in_flight=1` 时第二个并发请求必须在 `acquire_timeout` 后收到 `DeadlineExceeded`。
- R2：`insert_batch` 分块的 HTTP 层证据——5 行按 `max_rows_per_chunk=2` 必须产生 3 次独立 POST，而非拼成一次请求。
- 本轮**未变更**任何生产代码分支，仅补齐单测锚点与新增对抗性边界回归测试。
