# clickhousex 标准文档

**包名**：`clickhousex` v0.3.1 | **角色**：ClickHouse 分析型存储适配器 | **层级**：Production-default

---

## 1. 需求标准

### 目标

Clickhousex 提供 ClickHouse 分析型数据库的 HTTP 生产级适配器，支持批量 insert、有界连接池、AnalyticsSink trait 实现。

### 边界

- **包含**：HTTP 客户端（基于 reqwest）、批量 insert 写入器、有界连接池、AnalyticsSink 实现
- **不包含**：原生 TCP 协议（clickhouse-rs）、物化视图管理、ClickHouse Keeper
- **依赖**：reqwest、serde、serde_json、contracts、kernel

---

## 2. 设计标准

### 架构

```text
clickhousex
├── client      — ClickHouseHttpClient（HTTP 协议封装）
├── writer      — 批量写入器（INSERT INTO table FORMAT JSONEachRow）
├── pool        — 有界连接池管理
└── sink        — AnalyticsSink trait 实现
```text
