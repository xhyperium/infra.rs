# kafkax 0.3.2

| 字段 | 值 |
|------|-----|
| 日期 | 2026-07-22 |

## 变更摘要

- 冻结通用 EventBus AMO、单 owner 应用 ALO 与非原子 produce/checkpoint 重复窗口。
- 接入 rskafka rustls transport、webpki roots、可选 PEM CA 与 SASL/PLAIN。
- 远程明文、未知/不完整认证与非法环境配置在连接前 fail-closed。
- connect、metadata、topic 与 partition client 操作增加内部 deadline。

## 验证

```bash
cargo test -p kafkax --all-targets
cargo clippy -p kafkax --all-targets -- -D warnings
node scripts/kafka-broker-conformance.mjs
node scripts/kafka-tls-sasl-conformance.mjs
```

## 未承诺

SCRAM/OAuth/mTLS、consumer group/rebalance/fencing、HA、DLQ 与 native EOS 均为 NO-GO。
