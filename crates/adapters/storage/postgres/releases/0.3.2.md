| 字段 | 值 |
|------|-----|
| 日期 | 2026-07-22 |

## 变更摘要

- **事实更正**：0.3.2 只有 `resilience.rs` 的 `with_budget*` helper；从未交付
  `PostgresPool::with_retry_budget`、`execute_with_budget` 或 `query*_with_budget`。
- 这些历史 helper 未接收 `RetrySafety`，现归类为 unchecked compatibility；后续 round-03 候选
  新增显式 safety wrapper，但不追写成 0.3.2 已交付能力。
- live I/O 仍仅 `tests/live_postgres.rs`（`#[ignore]`）；默认 `--lib` 不碰外部 Postgres。

## 验证

```bash
cargo test -p postgresx --lib
cargo clippy -p postgresx --all-targets -- -D warnings
```
