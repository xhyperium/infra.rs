# postgresx 0.3.2

| 字段 | 值 |
|------|-----|
| 日期 | 2026-07-22 |
| 范围 | 内部 workspace（`publish = false`） |

## 变更摘要

- **生产入口接 resiliencx**：`PostgresPool::with_retry_budget` 后，`execute`/`query`/`query_one`/`query_opt` 经 `with_budget_async`。
- **显式 API**：`execute_with_budget` / `query_with_budget` / `query_one_with_budget` / `query_opt_with_budget`。
- **async helpers**：`with_budget_async` / `with_budget_async_noop`（`resilience.rs`）。
- live I/O 仍仅 `tests/live_postgres.rs`（`#[ignore]`）；默认 `--lib` 不碰外部 Postgres。

## 验证

```bash
cargo test -p postgresx --lib
cargo clippy -p postgresx --all-targets -- -D warnings
```
