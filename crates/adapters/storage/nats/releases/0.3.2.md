# natsx 0.3.2

| 字段 | 值 |
|------|-----|

## 变更摘要

- 冻结 Core AMO 与 JetStream durable pull/显式确认合同。
- Core/JetStream 命令增加内部 deadline；client/subscription capacity 与有限重连预算可配置。
- 远程 Disable/Prefer、URL userinfo、非法环境值与无界资源配置 fail-closed。
- connected/disconnected/slow-consumer stats 可观察；真实 broker 验证本地有界转发慢消费者。

## 验证

```bash
cargo test -p natsx --all-targets
cargo clippy -p natsx --all-targets -- -D warnings
node scripts/broker-conformance.mjs
```

## 重启恢复与未承诺

`scripts/nats-reconnect-conformance.mjs` 在同一 client、原 Core subscription、连接统计和
慢消费者场景连续 3 轮通过。Core 断线窗口仍不回放消息；超过有限重连预算后调用方必须
重建 client。Cluster/HA、自动 DLQ、NKey 与 KV/ObjectStore 全量未承诺。
