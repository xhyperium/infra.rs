# natsx 标准文档

**包名**：`natsx` v0.3.1 | **角色**：NATS 存储适配器（EventBus）| **层级**：Production-default

---

## 1. 需求标���

### 目标

natsx 提供 NATS 消息系统的 EventBus 适配器，支持 NATS Core（低延迟 pub/sub）和 JetStream（持久化/至少一次投递）。

### 边界

- **包含**：NatsPool（连接池）、NatsEventBus（implements EventBus）、JetStream 支持、TLS 配置
- **不包含**：Key-Value Store、Object Store、NATS Survey/Service
- **依赖**：async-nats、contracts、tokio、tracing、bytes

---

## 2. 设计标准

### 架构

```text
natsx
├── pool        — NatsPool（连接池管理）
├── nats_core   — NatsCoreEventBus（Core pub/sub）
├── jetstream   — NatsJetStreamEventBus（JetStream 持久化）
└── config      — 连接配置（URL、认证、TLS）
```text

## 5. 性能标准

| 操作 | 延迟目标 |
|------|----------|
| Core pub/sub | < 1ms |
| JetStream 持久化生产 | < 10ms |

## 6. 安全标准

## 7. 可靠性标准

- 消费者自动重连
- 流管理（StreamConfig 配置）
