# taosx 标准文档

**包名**：`taosx` v0.3.1 | **角色**：TDengine 时序数据库适配器 | **层级**：Production-default

---

## 1. 需求标准

### 目标

taosx 提供 TDengine 时序数据库的生产级 REST + WS 适配器，支持批量写入、时序查询、有界连接池。

### 边界

- **包含**：REST 客户端（基于 reqwest）、WS 检测（Native WS 协议判断）、批量写入器、有界连接池
- **不包含**：TDengine 原生 taosc 客户端、订阅功能、连续查询
- **依赖**：reqwest、tokio-tungstenite、serde、canonical、decimalx、contracts

---

## 2. 设计标准

### 架构

```text
taosx
├── rest        — REST 客户端（insert/query）
├── ws          — WS 检测 + 可选 Native WS 通信
├── writer      — 批量写入器（批量 insert）
└── pool        — 有界连接池管理
```text
