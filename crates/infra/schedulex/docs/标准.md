# schedulex 标准

**包名**：`schedulex` · **层级**：L1 internal · **依赖**：std-only

## 声明面

- `Scheduler`：ID registry，不执行 Job。
- `JobRunner`：宿主显式 `tick(now_ms)` 的进程内 deterministic runner。
- `Schedule`：Once、FixedDelay 与最小 cron 分钟子集。

## 正确性标准

- 非法 ID/调度在插入前 fail-closed。
- 同 tick 执行与 metadata 查询稳定排序。
- 时间回退不推进；大跨度不补跑。
- Job Err 继续后续 Job；panic 传播。
- 可见错误为简体中文。

## 安全与可靠性边界

无 unsafe、网络、文件系统、真实时钟或后台线程。进程崩溃后状态丢失属于设计边界；本 crate 不提供恢复、lease、分布式一致性或业务 SLA。

## 验收

以 public seam 测试、canonical coverage、clippy、rustdoc、API ratchet 与 SSOT 双镜像为证据；不使用未经测量的延迟数字或主观评分。
