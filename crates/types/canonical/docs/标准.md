# canonical 标准文档

**包名**：`canonical` v0.1.1 | **角色**：/types/ 跨层纯 DTO | **层级**：L2

---

## 1. 需求标准

### 目标

canonical 提供跨层纯 DTO 类型（ADR-001），包括 `Envelope<T>` 信封结构、Money 金额类型（复用 decimalx）。

### 边界

- **包含**：`Envelope<T>`（schema_version + payload）、Money、跨层 DTO 定义
- **不包含**：业务行为、序列化实现细节
- **依赖**：decimalx、serde

### 约束

- 纯 DTO — 无方法/行为，仅数据结构
- ADR-001 定义跨层通信契约
- 仅 L2 committed wire subset（v1-v1.3）

---

## 2. 设计标准

### 架构

```text
canonical
├── Envelope<T> { schema_version, payload }
└── Money { amount: Decimal, currency: CurrencyCode }
```text

### 数据流

```text
层A → Envelope::new(data) → 序列化 → 传输 → 反序列化 → 层B → Envelope.payload
```text
