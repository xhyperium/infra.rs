## 1. 需求标准

### 目标

evidence 提供审计证据的追加/查询/HMAC-SHA256 签名功能，支持可注入的远程传输。

### 边界

- **包含**：EvidenceAppender（追加）、EvidenceQuery（查询）、HMAC-SHA256 签名、FileEvidenceAppender、InMemoryEvidenceAppender、RemoteTransport trait
- **不包含**：合规产品、远程 CA、证据存储后端自动化管理
- **依赖**：sha2（仅生产依赖）

### 核心接口

| 符号 | 说明 |
|------|------|
| EvidenceAppender | 证据追加 trait |
| InMemoryEvidenceAppender | 内存实现 |
| FileEvidenceAppender | 文���实现 |
| EvidenceQuery | 证据查询 API |
| HmacSigner | HMAC-SHA256 签名器 |
