# contract-testkit 0.1.2

| 字段 | 值 |
|---|---|
| 日期 | 2026-07-22 |
| 范围 | 内部 test-support（`publish = false`） |

## 变更摘要

- 新增 ObjectStore、TimeSeriesStore、AnalyticsSink、PubSub 四个 portable core suite。
- EventBus portable surface 与 snapshot/replay Fake profile 分层。
- 补空标识、后端错误、对象不一致与查询缺失负向测试。
- OSS、TAOS、ClickHouse、Redis PubSub ignored target 接入真实 adapter 调用路径。

## 验证

```bash
cargo test -p contract-testkit --all-targets --all-features
cargo clippy -p contract-testkit --all-targets --all-features -- -D warnings
```

ignored target 默认未运行；本版本不把编译证据写成 live PASS。
