# contract-testkit 0.1.2

发布日期：2026-07-23（候选；合并与人工批准前不宣称已发布）

## 交付范围

- 为 ObjectStore、TimeSeriesStore、AnalyticsSink 与 PubSub 增加最小、诚实的 per-trait suite；
- 引入确定性 `FixtureNamespace`；
- 以 15 个 broken case 覆盖 14 个 trait；
- 接入 test-support production graph gate 与公开 API baseline。

## 非承诺

- 不提供真实网络、Docker、testnet、凭据或 live backend 证据；
- EventBus/PubSub 可移植 surface 不声明 delivery、replay、order、ack、backpressure 或次数；旧 EventBus Snapshot/Replay profile 仅为兼容入口；
- 不声明 ObjectStore/TimeSeriesStore 超出 suite 明示断言的后端语义；
- `BackendProfile` 只验证环境接线形状，不是 readiness attestation。

发布状态在人工审查、CI 与主干合并前保持候选。
