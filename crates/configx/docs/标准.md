# configx 标准文档

**包名**：`configx` v0.1.1 | **角色**：L1 配置存储 | **层级**：L1 Internal Ready

---

## 1. 需求标准

### 目标

configx 提供 L1 配置存储能力：内存 KV 存储 + 多源配置合并 + 配置变更 Watch + 密钥回写。

### 边界

- **包含**：内存 ConfigStore（KV 读写）、多源配置合并、配置变更 watch、密钥回写
- **不包含**：远程配置中心、文件配置源、系统环境变量读取
- **依赖**：仅 kernel（生产依赖）

### 核心接口

| 符号 | 说明 |
|------|------|
| ConfigStore | 配置存储 trait：get/set/watch/merge |
| MemoryConfigStore | 内存实现 |
| SecretRedaction | 密钥回写功能 |

### 约束

- 所有公开项必须文档
- 最小依赖策略（仅 kernel）

---

## 2. 设计标准

### 模块划分

| 模块 | 职责 |
|------|------|
| core | ConfigStore trait 定义 |
| memory | MemoryConfigStore 实现 |
| merge | 多源配置合并逻辑 |
| secret | 密钥回写与脱敏 |
| watch | 配置变更监听 |

### 数据流

```text
配置源1 + 配置源2 → merge → MemoryConfigStore → watch → 订阅方
                                           → get(key) → Value
                                           → set(key, value)
```
