# DDR-001: Rust 2024 Edition + MSRV 1.85

**日期**: 2026-07-21
**状态**: 已接受

## 上下文

项目初始化时需选定 Rust 工具链版本策略。Rust 2024 版号是最新稳定版号（2025 年 2 月发布），带来了多项语言改进：`unsafe` 作用域收紧、`impl Trait` 语法简化、`gen` 块预览等。

## 选项

| 选项 | 优点 | 缺点 |
| ------ | ------ | ------ |
| Edition 2021 + MSRV 1.70 | 最大兼容性，CI 镜像丰富 | 无法使用新版语言特性；与下游 xhyper.rs 不一致 |
| Edition 2021 + MSRV 1.85 | 更好的工具链支持 | 版号与 MSRV 版本号割裂 |
| **Edition 2024 + MSRV 1.85** | 最新语言特性；与下游 xhyper.rs 对齐 | 部分旧工具可能未适配 |

## 决策

采用 **Rust 2024 Edition + MSRV 1.85**。

## 后果

- **正面**: 与 `xhyperium/xhyper.rs` 保持工具链一致，减少跨仓库摩擦
- **正面**: 可使用新版号特性（如改进的 unsafe 语义）
- **负面**: 需要 CI 中专门配置 MSRV 验证 Job
- **负面**: 限制了部分旧 Rust 工具链用户的兼容性
