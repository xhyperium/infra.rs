## 格式

每份 ADR 遵循以下结构：

```markdown
# ADR-0001: 标题

- **状态**: proposed / accepted / deprecated / superseded
- **日期**: YYYY-MM-DD
- **上下文**: 描述需要决策的背景
- **决策**: 做出的决定及其理由
- **后果**: 决策带来的正面和负面影响
```

## 创建新 ADR

1. 复制模板到 `docs/decisions/ADR-NNNN.md`
2. 填写上下文、决策、后果
3. 提交 PR，在 PR 描述中引用本 ADR
