# DDR-004: 中文优先注释与文档

**日期**: 2026-07-21
**状态**: 已接受

## 上下文

项目团队以中文为母语。文档和注释的语言选择影响开发效率、代码审查速度和新成员上手成本。

## 选项

| 选项 | 优点 | 缺点 |
| ------ | ------ | ------ |
| 全英文 | 国际化标准，开源社区惯用 | 团队表达精度下降；审查效率降低 |
| **中文优先** | 团队认知负担最低；表达精确 | 非中文贡献者门槛高；工具链支持需验证 |
| 中英混用 | 灵活 | 不一致性高，风格难以统一 |

## 决策

**中文优先**: 代码注释、项目文档、用户可见错误信息使用中文。技术术语保留英文。标识符使用英文（Rust 惯例）。英文技术文档遵循 ASD-STE100 规范。LICENSE 保留原文。

## 后果

- **正面**: 团队内部沟通和代码审��效率显著提升
- **正面**: 认知负担降低，减少翻译损耗
- **负面**: 非中文社区成员贡献门槛提高
- **负面**: 部分文档工具（拼写检查）需额外配置
