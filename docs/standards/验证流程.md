
### 2. 拼写检查

```bash
npx cspell "crates/**/标准.md" "tools/**/标准.md"
```

**期望**：`Issues found: 0 in 0 files`

拼写词典由 `.cspell.json` 管理，分为两组：

- `infra-rs`：所有 crate 名、项目特有缩写（SSOT、HMAC、PITR 等）
- `rust-crates`：Rust 生态关键词（tokio、serde、clippy 等）

误报补充：编辑 `.cspell.json` → `dictionaryDefinitions[].words` 数组追加。

### 3. 17 项标准完整性

```python
import glob

STANDARDS = [
    "需求标准", "设计标准", "编码标准", "测试标准", "性能标准",
    "安全标准", "可靠性标准", "可维护性标准", "数据标准",
    "风控标准", "回测标准", "上线标准", "运维标准",
    "文档标准", "交付标准", "评分标准", "验收标准",
]

for f in glob.glob("crates/**/标准.md", recursive=True) + glob.glob("tools/**/标准.md", recursive=True):
    with open(f) as fh:
        text = fh.read()
    present = [s for s in STANDARDS if s in text]
    missing = [s for s in STANDARDS if s not in text]
    status = "OK" if not missing else f"MISSING: {missing}"
    print(f"{status}\t{f}")
```

**期望**：全部 24 份文档命中 17/17 项标准。不适用标准须标注 `N/A` 并说明原因。

### 4. 代码块结构——检测

```bash
python3 scripts/standards/check-fences.py
```

**期望**：`24 文件，24 OK，0 BROKEN`

检测规则：

- 奇数代码块（开）：必须含语言标签 `text`
- 偶数代码块（闭）：必须为纯 ` ``` `，无多余信息
- 文件末尾：偶数个代码块（已闭合）

### 5. 代码块结构——修复

```bash
python3 scripts/standards/fix-fences.py
```

在检测发现 BROKEN 文件后运行。自动修正所有不匹配的开闭标记，保持 `markdownlint` 和 `check-fences.py` 双��校验通过。

### 6. Workspace 品质门禁

```bash
# 整体门禁
node scripts/quality-gates/check.mjs

# 依赖集中管理
node scripts/quality-gates/check-workspace-deps.mjs
```

**期望**：全部 PASS。标准文档不引入新的第三方依赖，不改变 crate 结构。

> 当前 `check.mjs` 不包含标准文档专项门禁。如需 CI 级别拦截 fence 错误，可在 `check.mjs` 中增加 `execSync("python3 scripts/standards/check-fences.py")`。

### 7. 链接有效性

CI 自动执行的 `链接检查` 门禁会扫描文档中的超链接。如新增外部链接，需确保：

```bash
# 本地抽检新链接
grep -oP 'https?://[^\s)]+' 路径/标准.md | while read url; do
  curl -sfo /dev/null -w "%{http_code} $url\n" "$url" || echo "DEAD: $url"
done
```

**期望**：无损坏链接（HTTP 200/30x）。

## 排查指引

| 问题 | 根因 | 排查步骤 |
|------|------|---------|
| MD040 大量报错 | 代码块开闭标记未交替 | 运行 `python3 scripts/standards/check-fences.py` 定位，再运行 `python3 scripts/standards/fix-fences.py` 修复 |
| MD022/MD032 | 标题/列表前后缺空行 | `npx markdownlint-cli2 --fix` 自动添加 |
| MD033 inline HTML | 泛型 `<T>` 被解析为 HTML | 在泛型两侧加反引号：`Envelope<T>` 而非 `Envelope<T>` |
| 拼写误报过多 | 词典未收录项目术语 | 编辑 `.cspell.json` → `dictionaryDefinitions[0].words` 追加 |
| `.cspell.json` 不存在 | 未初始化 | 从项目根 `.cspell.json` 模板复制，或参考 CI 使用的配置 |
| 标准缺失（未达 17/17） | 文档未按模板撰写 | 对照 `docs/standards/标准文档模板.md` 补充对应节 |
| CI fence 检查未拦截 | `check.mjs` 未集成 | 在 `check.mjs` 中增加 `execSync("python3 scripts/standards/check-fences.py")` |
