## 3. 编码标准

### 命名

- 类型：UpperCamelCase
- 函数/方法：snake_case
- 模块：snake_case
- 常量：SCREAMING_SNAKE_CASE

### 风格

- 遵循 `cargo fmt` 默认风格
- `#[must_use]` 标注所有返回 Result 的函数
- 公开 API 必须提供完整 rustdoc

### 错误处理

- 使用 `XError` 作为 crate 公开错误类型
- 内部错误使用 `thiserror` / 自定义 ErrorKind
- 所有 fallible 函数返回 `XResult<T>` 或专用 Result 类型

### 志

- 关键路径：`tracing::info!`
- 调试：`tracing::debug!`
- 异常：`tracing::warn!` / `tracing::error!`

### 释

- 公开项：完整 rustdoc（说明、Panics、Errors、Examples）
- 内项：仅复杂逻辑需要注释
- 中文注释（治理约定）

- `cargo deny` 检查允许列表
